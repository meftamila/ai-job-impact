# AI Job Impact 2030

A comprehensive analysis and interactive dashboard exploring the projected impact of artificial intelligence on employment sectors by 2030.

## 📊 Project Overview

This project provides data-driven insights into how AI is expected to influence different job markets and employment sectors through 2030. It combines statistical analysis, predictive modeling, and interactive visualization to help stakeholders understand AI-related workforce trends.

![AI Impact on Jobs 2030 Dashboard](https://github.com/meftamila/ai-job-impact/blob/main/AI_Impact_on_Jobs_2030.xlsx)

## 📁 Repository Contents

- **`AI_Impact_on_Jobs_2030.xlsx`** - Excel dashboard with interactive visualizations and data summaries of job impact projections
- **`job_prediction.ipynb`** - Jupyter Notebook containing comprehensive data analysis, statistical modeling, and AI job impact predictions
- **`job_risk_prediction.py`** - Python module for predicting job risk levels based on AI automation potential
- **`ai_jobs_2030_website.html`** - Interactive HTML dashboard for web-based exploration of the findings

## 🛠️ Technology Stack

- **Jupyter Notebook** (85.6%) - Interactive data analysis and visualization
- **HTML** (11.4%) - Web-based dashboard interface
- **Python** (3%) - Predictive modeling and analysis utilities
